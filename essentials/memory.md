## 🏆 Top 3 推荐

### 1. self-improving-agent ⭐⭐⭐⭐⭐

**定位：** 让 Agent 持续学习和成长

**一句话：** 唯一带"复利效应"的 Skill，必装首选

**推荐理由：**
- 🔄 **复利效应** - 每次错误都变成经验
- 📈 **效率提升** - 30 天后效率提升 40%+
- 🤖 **零维护** - 安装后自动运行
- 🔍 **透明可查** - 可查看学习记录

**真实效果（30 天）：**
- 积累 150+ 条经验教训
- 重复错误减少 80%
- 任务完成时间缩短 40%
- 累计节省 36 小时

**安装：**
```bash
clawhub install self-improving-agent
```

**配置：** ⭐ 简单（零配置）

**详细报告：** [30 天使用报告](../examples/self-improving-agent-30days.md)

---

### 2. starmemo ⭐⭐⭐⭐

**定位：** 结构化记忆 + 知识库

**一句话：** 知识有序存储，快速检索

**推荐理由：**
- 🧠 **结构化存储** - 知识有序管理
- 🔍 **语义搜索** - 快速找到所需
- 🤖 **AI 优化** - 自动整理和关联
- 📊 **知识图谱** - 可视化知识关系

**典型场景：**
- 记录会议要点
- 整理学习笔记
- 管理项目文档
- 建立个人知识库

**安装：**
```bash
clawhub install starmemo
```

**配置：** ⭐⭐ 中等

---

### 3. summarize ⭐⭐⭐⭐

**定位：** 多格式文档摘要

**一句话：** 快速提取核心信息，节省阅读时间

**推荐理由：**
- 📄 **多格式支持** - PDF/图片/音频/URL
- 🤖 **AI 摘要** - 自动生成精炼摘要
- ⚡ **快速处理** - 大文档也能快速处理
- 💾 **节省时间** - 节省 80% 阅读时间

**典型场景：**
- 快速阅读论文
- 提取会议记录要点
- 总结长文档
- 整理访谈内容

**安装：**
```bash
clawhub install summarize
```

**配置：** ⭐ 简单
