## 🏆 Top 5 推荐

### 1. searxng ⭐⭐⭐⭐⭐

**定位：** 日常搜索主力

**一句话：** 隐私保护的本地元搜索引擎，无需 API Key

**推荐理由：**
- 🔒 隐私保护，不记录搜索历史
- 💰 完全免费，自建实例无配额限制
- 🔍 多源聚合（Google/Bing/DuckDuckGo 等）
- 📱 日常使用频率最高

**安装：**
```bash
clawhub install searxng
```

**配置：** ⭐ 简单（2 分钟）

---

### 2. tavily-search ⭐⭐⭐⭐⭐

**定位：** 深度研究

**一句话：** AI 优化的实时网络搜索，结果更精准

**推荐理由：**
- 🤖 AI 优化，结果精准
- 📊 自动生成摘要
- 🔍 适合竞品分析/市场调研
- ⚡ 速度快

**安装：**
```bash
clawhub install tavily-search
```

**配置：** ⭐⭐ 中等（需 API Key，有免费额度）

---

### 3. baidu-search ⭐⭐⭐⭐

**定位：** 中文内容搜索

**一句话：** 百度 AI 搜索引擎集成，中文内容覆盖全面

**推荐理由：**
- 🇨🇳 中文友好，百度生态
- 🤖 AI 优化搜索
- 📱 移动端适配
- 🔍 多模态搜索（文本/图片/视频）

**安装：**
```bash
clawhub install baidu-search
```

**配置：** ⭐ 简单

---

### 4. github-trending ⭐⭐⭐⭐

**定位：** 开源项目发现

**一句话：** GitHub Trending 热门项目追踪

**推荐理由：**
- 📊 发现热门开源项目
- 🔥 追踪技术趋势
- 💡 寻找优质项目参考
- ⏰ 支持今日/每周/每月筛选

**安装：**
```bash
clawhub install github-trending
```

**配置：** ⭐ 简单（无需 API Key）
