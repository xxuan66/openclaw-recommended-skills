## 🏆 Top 3 推荐

### 1. summarize ⭐⭐⭐⭐⭐

**定位：** 多格式文档摘要

**一句话：** 快速提取核心信息，节省 80% 阅读时间

**推荐理由：**
- 📄 **多格式支持** - PDF/图片/音频/URL 全覆盖
- 🤖 **AI 摘要** - 自动生成精炼摘要
- ⚡ **快速处理** - 大文档也能快速处理
- 💾 **节省时间** - 快速了解核心内容

**典型场景：**
- 快速阅读行业报告
- 提取论文核心观点
- 总结会议记录
- 整理访谈内容

**安装：**
```bash
clawhub install summarize
```

**配置：** ⭐ 简单

---

### 2. xiaohongshu-mcp ⭐⭐⭐⭐

**定位：** 小红书内容自动化

**一句话：** 内容发布、搜索、分析一站式管理

**推荐理由：**
- 📱 **内容发布** - 自动发布图文/视频
- 🔍 **内容搜索** - 追踪热门话题
- 📊 **数据分析** - 查看笔记详情和评论
- 👤 **用户管理** - 管理个人主页和内容流

**典型场景：**
- 批量发布内容
- 追踪热门话题
- 分析竞品笔记
- 管理多个账号

**安装：**
```bash
clawhub install xiaohongshu-mcp
```

**配置：** ⭐⭐⭐ 复杂（需 Cookie 配置）

---

### 3. douyin-hot-trend ⭐⭐⭐⭐

**定位：** 抖音热榜数据获取

**一句话：** 追踪热门视频、挑战赛、音乐趋势

**推荐理由：**
- 🔥 **热榜数据** - 实时获取抖音热搜
- 📊 **热度分析** - 查看热度值和趋势
- 🎵 **多领域** - 视频/挑战赛/音乐全覆盖
- 🔗 **跳转链接** - 直接访问热门内容

**典型场景：**
- 追踪热门话题
- 寻找创作灵感
- 分析内容趋势
- 竞品监控

**安装：**
```bash
clawhub install douyin-hot-trend
```

**配置：** ⭐ 简单
