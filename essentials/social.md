## 🏆 Top 3 推荐

### 1. xiaohongshu-mcp ⭐⭐⭐⭐⭐

**定位：** 小红书内容自动化

**一句话：** 内容发布、搜索、分析一站式管理

**推荐理由：**
- 📱 **内容发布** - 自动发布图文/视频
- 🔍 **内容搜索** - 追踪热门话题和笔记
- 📊 **数据分析** - 查看笔记详情、评论和互动
- 👤 **用户管理** - 管理个人主页和内容流
- 🎯 **精准运营** - 基于数据优化内容策略

**典型场景：**
- 批量发布内容（图文/视频）
- 追踪热门话题和标签
- 分析竞品笔记数据
- 管理多个账号内容
- 自动回复评论

**安装：**
```bash
clawhub install xiaohongshu-mcp
```

**配置：** ⭐⭐⭐ 复杂（需 Cookie 配置）

**详细文档：** [使用指南](../skills/xiaohongshu-mcp/README.md)

---

### 2. douyin-hot-trend ⭐⭐⭐⭐

**定位：** 抖音热榜数据获取

**一句话：** 追踪热门视频、挑战赛、音乐趋势

**推荐理由：**
- 🔥 **热榜数据** - 实时获取抖音热搜榜
- 📊 **热度分析** - 查看热度值和排名变化
- 🎵 **多领域** - 视频/挑战赛/音乐全覆盖
- 🔗 **跳转链接** - 直接访问热门内容
- 📈 **趋势追踪** - 监控热度变化

**典型场景：**
- 追踪热门话题和标签
- 寻找内容创作灵感
- 分析内容趋势
- 竞品监控
- 制定发布策略

**安装：**
```bash
clawhub install douyin-hot-trend
```

**配置：** ⭐ 简单（无需 API Key）

---

### 3. github-trending ⭐⭐⭐⭐

**定位：** GitHub 趋势追踪

**一句话：** 技术圈热点追踪，发现热门开源项目

**推荐理由：**
- 📊 **技术热点** - 追踪开发者关注的项目
- 🔥 **趋势分析** - 今日/每周/每月热门
- 💡 **灵感来源** - 发现优质项目参考
- 🌍 **多语言** - 按编程语言筛选
- 🚀 **早期发现** - 找到潜力项目

**典型场景：**
- 追踪技术趋势
- 寻找开源项目参考
- 发现新技术/框架
- 竞品分析
- 技术选型参考

**安装：**
```bash
clawhub install github-trending
```

**配置：** ⭐ 简单（无需 API Key）
