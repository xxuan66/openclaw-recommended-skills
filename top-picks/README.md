# 🏆 Top Picks - 月度精选 Skill

> 每月只推荐 5 个最值得安装的 Skill，宁缺毋滥

---

## 📌 这是什么？

**Top Picks** 是「推荐清单中的推荐清单」。

如果说 `skills-index.json` 是电话黄页（全面但需要筛选），那么 Top Picks 就是米其林指南（精选且带深度评测）。

---

## 🎯 和推荐清单的区别

| 维度 | 推荐清单 | Top Picks |
|------|---------|-----------|
| **定位** | 全面索引 | 精选榜单 |
| **数量** | 14+ 个（持续增加） | 固定 5 个 |
| **内容** | 基础信息 + 安装命令 | 深度评测 + 使用案例 |
| **更新** | 每日新增 | 每月更新 |
| **目的** | 查找"有什么" | 决策"装什么" |

## 🏆 评选标准

### 入围条件
1. ✅ 已在 ClawHub 上架
2. ✅ 实际使用超过 7 天
3. ✅ 无重大 Bug 或安全问题
4. ✅ 文档完整

### 评分维度
| 维度 | 权重 | 说明 |
|------|------|------|
| **实用性** | 30% | 是否解决真实问题 |
| **稳定性** | 25% | 是否可靠、少 Bug |
| **易用性** | 20% | 配置是否简单 |
| **文档质量** | 15% | 文档是否清晰完整 |
| **长期价值** | 10% | 是否值得长期使用 |

### 评选流程
```
1. 候选提名（来自推荐清单）
   ↓
2. 实际测试（至少 7 天）
   ↓
3. 评分排序（按权重计算）
   ↓
4. Top 5 入选
   ↓
5. 撰写深度评测
   ↓
6. 发布到 top-picks/
```

## 💡 如何使用

### 新手（第一次用 OpenClaw）
```bash
# 直接安装 Top 5，闭眼不踩坑
clawhub install self-improving-agent
clawhub install searxng
clawhub install github
clawhub install skill-vetter
clawhub install tavily-search
```

### 进阶（已有基础配置）
1. 查看本月 Top 5 榜单
2. 阅读详细评测
3. 根据需求选择性安装

### 老手（想发现新东西）
- 关注每月 1 日的更新
- 查看 [全部推荐清单](../README.md)
- 参与提名和评选
