## 🥇 Top 1：self-improving-agent

**一句话推荐：** 让你的 Agent 越用越聪明，唯一值得"投资"的 Skill

| 指标 | 说明 |
|------|------|
| **版本** | 3.0.0 |
| **类别** | 🧠 学习/记忆 |
| **安装** | `clawhub install self-improving-agent` |
| **配置难度** | ⭐ 简单 |
| **长期价值** | ⭐⭐⭐⭐⭐ |

### 为什么排第一？

1. **复利效应** - 每次错误都变成经验，30 天后 Agent 明显变聪明
2. **零维护** - 安装后自动运行，无需额外配置
3. **通用性强** - 所有场景都受益，不是特定功能
4. **开源透明** - 可以看到 Agent 学到了什么

### 真实效果

```markdown
用户 A（使用 30 天）：
- 积累 150+ 条经验教训
- 重复错误减少 80%
- 任务完成时间缩短 40%
```

### 快速开始

```bash
# 安装
clawhub install self-improving-agent

# 查看学到的内容
cat ~/.openclaw/workspace/memory/lessons_learned.md

# 查看每日总结
cat ~/.openclaw/workspace/memory/2026-03-13.md
```

### 适合人群
- ✅ 所有 OpenClaw 用户（100% 推荐）
- ✅ 希望 Agent 持续成长的用户
- ✅ 经常重复类似任务的用户

## 🥈 Top 2：searxng

**一句话推荐：** 隐私搜索刚需，无需 API Key，日常使用频率最高

| 指标 | 说明 |
|------|------|
| **版本** | 1.0.3 |
| **类别** | 🔍 搜索 |
| **安装** | `clawhub install searxng` |
| **配置难度** | ⭐⭐ 中等（需部署 SearXNG 实例） |
| **使用频率** | ⭐⭐⭐⭐⭐ |

### 核心优势

1. **隐私保护** - 不记录搜索历史，不追踪用户
2. **多源聚合** - 同时搜索 Google、Bing、DuckDuckGo 等
3. **无需 API Key** - 自建实例，无配额限制
4. **可定制** - 可选择搜索源和过滤规则

### 典型使用场景

```bash
# 日常搜索
openclaw agent -m "搜索 Python 异步编程最佳实践"

# 指定搜索源
openclaw agent -m "在 GitHub 搜索 openclaw 相关项目"

# 避免上下文干扰（独立会话）
openclaw agent -m "搜索 AI Agent 框架对比" --session-id search-$(date +%s)
```

### 适合人群
- ✅ 注重隐私的用户
- ✅ 高频搜索用户
- ✅ 需要多源验证信息的用户

## 🏅 Top 5：tavily-search

**一句话推荐：** AI 优化的搜索，结果更精准，适合研究场景

| 指标 | 说明 |
|------|------|
| **版本** | 1.0.0 |
| **类别** | 🔍 搜索 |
| **安装** | `clawhub install tavily-search` |
| **配置难度** | ⭐⭐ 需 API Key |
| **搜索质量** | ⭐⭐⭐⭐⭐ |

### 与 searxng 的区别

| 特性 | searxng | tavily-search |
|------|---------|---------------|
| **定位** | 日常搜索 | 研究级搜索 |
| **结果** | 原始链接 | AI 摘要 + 链接 |
| **API** | 自建免费 | 需 API Key（有免费额度） |
| **速度** | 快 | 中等 |
| **精度** | 中 | 高 |

### 典型使用场景

```bash
# 深度研究
openclaw agent -m "调研 2026 年 AI Agent 框架发展趋势"

# 竞品分析
openclaw agent -m "分析 OpenClaw 的主要竞争对手"
```

### 适合人群
- ✅ 需要做深度研究的用户
- ✅ 内容创作者
- ✅ 市场/竞品分析人员
