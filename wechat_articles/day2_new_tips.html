⚡ 5 个效率小技巧

让你的 OpenClaw 效率翻倍！

2026-03-11 · Day 02

💡 前言

使用 OpenClaw 一段时间后，掌握一些小技巧可以让你的效率翻倍提升！

今天分享 5 个实用技巧，每个都能让你事半功倍！

1️⃣ 使用 --session-id 避免上下文干扰

🎯 使用场景：

当你需要执行独立任务（如搜索、查询）时，不希望受到之前对话上下文的影响。

💻 使用方法：

# 每次使用不同的 session-id
openclaw agent -m "搜索 Python 教程" \
  --session-id search-$(date +%s)

# 或者使用固定 ID 保持会话
openclaw agent -m "继续昨天的工作" \
  --session-id my-project-001

✅ 好处：

  • ✅ 避免上下文污染
  • ✅ 搜索结果更精准
  • ✅ 可以并行多个任务

2️⃣ 快速切换模型

🎯 使用场景：

不同任务需要不同模型：简单任务用便宜模型，复杂任务用高质量模型。

💻 使用方法：

# 切换到备用模型
openclaw models set <provider>/<model-name>

# 执行任务
openclaw agent -m "复杂任务..."

# 切回默认模型
openclaw models set <default-model>

💡 进阶技巧：

# 创建切换脚本
cat > switch-model.sh << 'EOF'
openclaw models set <provider>/<model-name>
echo "✅ 已切换到备用模型"
EOF

cat > switch-default.sh << 'EOF'
openclaw models set <default-model>
echo "✅ 已切回默认模型"
EOF

chmod +x switch-*.sh
./switch-model.sh

3️⃣ 批量安装 Skill

🎯 使用场景：

初次安装或重装系统后，需要快速安装多个 Skill。

💻 使用方法：

# 一键安装 TOP10 Skill
for skill in searxng self-improving-agent \
  tavily-search github-trending \
  summarize weather find-skills; do
  echo "📦 安装 $skill..."
  clawhub install $skill
done

echo "✅ 全部安装完成！"

✅ 好处：

  • ✅ 一行命令安装多个
  • ✅ 自动处理依赖
  • ✅ 节省时间

4️⃣ 配置备份与恢复

🎯 使用场景：

更换设备或重装系统时，需要快速恢复配置。

💻 使用方法：

# 备份配置
cp ~/.openclaw/openclaw.json \
   openclaw-backup-$(date +%Y%m%d).json

# 恢复配置
cp openclaw-backup-20260311.json \
   ~/.openclaw/openclaw.json

# 或者使用 git 管理
cd ~/.openclaw
git add openclaw.json
git commit -m "备份配置"
git push

✅ 好处：

  • ✅ 防止配置丢失
  • ✅ 快速恢复环境
  • ✅ 版本管理配置

5️⃣ 使用 cron 自动化

🎯 使用场景：

定时执行任务：每日提醒、自动备份、模型切换等。

💻 使用方法：

# 添加每日提醒
openclaw cron add \
  --name "每日提醒" \
  --cron "0 9 * * *" \
  --message "早上好！今天有什么计划？"

# 查看任务列表
openclaw cron list

# 手动测试
openclaw cron run <task-id>

✅ 好处：

  • ✅ 自动执行重复任务
  • ✅ 不会忘记重要事项
  • ✅ 解放双手
