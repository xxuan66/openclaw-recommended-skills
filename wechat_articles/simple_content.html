🦞 OpenClaw 必装 Skill TOP10

精选 10 个超实用的 OpenClaw Skill

🎯 前言

OpenClaw 是一个强大的本地 AI 助手框架，而 Skill 是它的核心扩展能力。

今天为大家精选了 10 个必装 Skill！

🥇 TOP1 - searxng

类别：搜索 | 评分：⭐⭐⭐⭐⭐

安装：clawhub install searxng

  • ✅ 隐私优先，不追踪用户
  • ✅ 本地部署，数据不出境
  • ✅ 完全免费

🥈 TOP2 - self-improving-agent

类别：学习/记忆 | 评分：⭐⭐⭐⭐⭐

安装：clawhub install self-improving-agent

  • ✅ 自动从错误中学习
  • ✅ 持续改进能力
  • ✅ 越用越聪明

🥉 TOP3 - github

类别：开发 | 评分：⭐⭐⭐⭐⭐

安装：clawhub install github

  • ✅ 完整的 GitHub API 支持
  • ✅ PR/Issue/CI 一站式管理
  • ✅ 开发者必备

🏆 其他热门推荐

4️⃣ tavily-search - AI 优化的实时搜索 ⭐⭐⭐⭐⭐

安装：clawhub install tavily-search

5️⃣ skill-vetter - 安全审查工具 ⭐⭐⭐⭐⭐

安装：clawhub install skill-vetter

6️⃣ summarize - 多格式文档摘要 ⭐⭐⭐⭐

安装：clawhub install summarize

7️⃣ gog - Google Workspace 集成 ⭐⭐⭐⭐

安装：clawhub install gog

8️⃣ starmemo - 智能记忆系统 ⭐⭐⭐⭐

安装：clawhub install starmemo

9️⃣ baidu-search - 百度搜索集成 ⭐⭐⭐⭐

安装：clawhub install baidu-search

🔟 find-skills - Skill 发现助手 ⭐⭐⭐⭐

安装：clawhub install find-skills

📦 快速安装

for skill in searxng self-improving-agent github tavily-search skill-vetter summarize gog starmemo baidu-search find-skills; do
  clawhub install $skill
done
