## 🏆 TOP10 推荐榜单

### 1️⃣ searxng - 隐私保护的元搜索引擎

**评分：** ⭐⭐⭐⭐⭐  
**类别：** 搜索  
**安装：** `clawhub install searxng`

**推荐理由：**
- ✅ 隐私优先，不追踪用户
- ✅ 本地部署，数据不出境
- ✅ 聚合多个搜索引擎结果
- ✅ 完全免费

**使用场景：** 日常搜索、技术研究、信息查询

## 📦 快速安装

### 批量安装全部推荐

```bash
# 一键安装 TOP10
for skill in searxng self-improving-agent github tavily-search skill-vetter summarize gog starmemo baidu-search find-skills; do
  clawhub install $skill
done
```

### 按需安装

```bash
# 只安装搜索类
clawhub install searxng
clawhub install tavily-search
clawhub install baidu-search

# 只安装学习类
clawhub install self-improving-agent
clawhub install starmemo
```
