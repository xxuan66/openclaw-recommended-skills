让你的 OpenClaw 效率翻倍！

💡 前言

使用 OpenClaw 一段时间后，掌握一些小技巧可以让你的效率翻倍提升！

今天分享 5 个实用技巧，每个都能让你事半功倍！

1️⃣ 使用 --session-id 避免上下文干扰

🎯 使用场景：

当你需要执行独立任务（如搜索、查询）时，不希望受到之前对话上下文的影响。

💻 使用方法：

# 每次使用不同的 session-id
openclaw agent -m "搜索 Python 教程" \
  --session-id search-$(date +%s)

# 或者使用固定 ID 保持会话
openclaw agent -m "继续昨天的工作" \
  --session-id my-project-001

✅ 好处：

  • ✅ 避免上下文污染
  • ✅ 搜索结果更精准
  • ✅ 可以并行多个任务

2️⃣ 快速切换模型

🎯 使用场景：

不同任务需要不同模型：简单任务用便宜模型，复杂任务用强力模型。

💻 使用方法：

# 切换到备用模型（如 mimo）
openclaw models set mimo/mimo-claw-0301

# 执行任务
openclaw agent -m "复杂任务..."

# 切回默认模型（qwen）
openclaw models set bailian/qwen3.5-plus

💡 进阶技巧：

# 创建切换脚本
cat > switch-mimo.sh << 'EOF'
openclaw models set mimo/mimo-claw-0301
echo "✅ 已切换到 mimo"
EOF

cat > switch-qwen.sh << 'EOF'
openclaw models set bailian/qwen3.5-plus
echo "✅ 已切换到 qwen"
EOF

chmod +x switch-*.sh
./switch-mimo.sh

3️⃣ 批量安装 Skill

🎯 使用场景：

初次安装或重装系统后，需要快速安装多个 Skill。

💻 使用方法：

# 一键安装 TOP10 Skill
for skill in searxng self-improving-agent \
  github tavily-search skill-vetter \
  summarize gog starmemo baidu-search \
  find-skills; do
  clawhub install $skill
done

echo "✅ 全部安装完成！"

💡 进阶技巧：

# 按类别安装
# 只安装搜索类
for skill in searxng tavily-search baidu-search; do
  clawhub install $skill
done

# 只安装学习类
for skill in self-improving-agent starmemo; do
  clawhub install $skill
done

4️⃣ 配置备份与恢复

🎯 使用场景：

重装系统、迁移服务器或防止配置丢失。

💻 使用方法：

# 备份配置
cp ~/.openclaw/openclaw.json \
   ~/.openclaw/openclaw.json.bak.$(date +%Y%m%d)

# 备份整个配置目录
tar -czf openclaw-backup-$(date +%Y%m%d).tar.gz \
    ~/.openclaw/

# 恢复配置
cp ~/.openclaw/openclaw.json.bak.20260310 \
   ~/.openclaw/openclaw.json

# 重启 Gateway
openclaw gateway restart

💡 自动化备份：

# 添加到 crontab，每周日备份
0 2 * * 0 tar -czf ~/openclaw-backup-$(date +\%Y\%m\%d).tar.gz ~/.openclaw/

5️⃣ 使用 cron 实现自动化

🎯 使用场景：

定时执行任务，如每日更新、定时提醒、自动发布等。

💻 使用方法：

# 添加定时任务
openclaw cron add \
  --name "每日更新" \
  --cron "0 6 * * *" \
  --message "执行更新任务"

# 查看任务列表
openclaw cron list

# 删除任务
openclaw cron remove --id

💡 实际应用：

# 每天早上 6 点更新 GitHub
openclaw cron add \
  --name "GitHub 更新" \
  --cron "0 6 * * *" \
  --message "更新 GitHub 项目"

# 每天早上 9 点生成公众号内容
openclaw cron add \
  --name "公众号内容" \
  --cron "0 9 * * *" \
  --message "生成公众号文章"

# 每天中午 11 点提醒发布
openclaw cron add \
  --name "发布提醒" \
  --cron "0 11 * * *" \
  --message "提醒发布文章"
