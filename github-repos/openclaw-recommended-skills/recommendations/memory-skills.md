# 🧠 学习/记忆类 Skill 推荐

## 1. self-improving-agent

**版本:** 3.0.0  
**Slug:** `self-improving-agent`  
**类别:** 学习/记忆  

### 简介
自我反思 + 自我批评 + 自我学习 + 自组织记忆系统。Agent 会评估自己的工作，发现错误，并持续改进。

### 推荐理由
- ✅ 自动从错误中学习
- ✅ 结构化记忆管理
- ✅ 持续改进能力
- ✅ 适合长期任务

### 安装命令
```bash
clawhub install self-improving-agent
```

### 配置要求
- 自动创建 `.learnings/` 目录
- 自动提升到 AGENTS.md/TOOLS.md

### 使用场景
1. 用户纠正你时
2. 命令/操作失败时
3. 发现更好的方法时
4. 知识更新时

## 2. starmemo

**版本:** 2.0.0  
**Slug:** `starmemo`  
**类别:** 记忆/知识库  

### 简介
智能记忆系统 v2.0 - 结构化记忆 + 知识库 + 启发式召回 + AI 优化。

### 推荐理由
- ✅ 结构化记忆存储
- ✅ 智能召回
- ✅ AI 优化记忆
- ✅ 知识库管理

### 安装命令
```bash
clawhub install starmemo
```

### 配置要求
- 需要配置记忆存储路径

### 使用场景
1. 长期项目记忆
2. 用户偏好存储
3. 领域知识积累
4. 跨会话上下文
