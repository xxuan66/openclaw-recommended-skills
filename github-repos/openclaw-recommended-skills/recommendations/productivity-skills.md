# 📧 办公/效率类 Skill 推荐

## 1. gog

**版本:** 1.0.0  
**Slug:** `gog`  
**类别:** 办公/效率  

### 简介
Google Workspace CLI，支持 Gmail、Calendar、Drive、Contacts、Sheets、Docs。

### 推荐理由
- ✅ 全套 Google 集成
- ✅ 命令行操作
- ✅ 自动化友好
- ✅ 免费使用

### 安装命令
```bash
clawhub install gog
```

### 配置要求
- 需要 Google OAuth 认证
- 首次使用需授权

### 支持服务
| 服务 | 功能 |
|------|------|
| Gmail | 邮件管理 |
| Calendar | 日历事件 |
| Drive | 文件管理 |
| Contacts | 联系人 |
| Sheets | 表格 |
| Docs | 文档 |

### 使用示例
```bash
# 查看日历
gog calendar list

# 发送邮件
gog gmail send --to xxx@example.com --subject "Hello"

# 列出 Drive 文件
gog drive list
```

## 2. find-skills

**版本:** 0.1.0  
**Slug:** `find-skills`  
**类别:** 效率/发现  

### 简介
帮你发现和安装新的 Agent Skill。当用户问"如何做 X"或"有 X 的 skill 吗"时使用。

### 推荐理由
- ✅ 发现新技能
- ✅ 扩展能力
- ✅ 自动搜索
- ✅ 持续更新

### 安装命令
```bash
clawhub install find-skills
```

### 配置要求
- 无需额外配置

### 使用场景
1. 用户询问新功能
2. 寻找特定类型 skill
3. 扩展 Agent 能力
4. 探索 ClawHub
