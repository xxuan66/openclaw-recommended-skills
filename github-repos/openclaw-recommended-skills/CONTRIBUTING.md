# 🤝 贡献指南

感谢你对 OpenClaw 推荐 Skill 清单的贡献！

## 如何贡献

### 推荐新 Skill

1. Fork 本仓库
2. 在 `recommendations/` 目录下创建新的推荐文件
3. 使用以下格式：

```markdown
### Skill 名称

- **Slug:** `skill-slug`
- **版本:** 1.0.0
- **类别:** 搜索/学习/开发/安全/办公/内容/...
- **简介:** 一句话描述
- **推荐理由:** 为什么值得安装（3-5 点）
- **安装命令:** `clawhub install skill-slug`
- **配置要求:** 需要什么环境/Token
- **使用场景:** 什么情况下使用
```

4. 同时更新 `skills-index.json` 文件
5. 提交 PR

### 更新现有 Skill

- 更新版本号
- 更新描述
- 添加新的使用场景
- 修复过时信息

### 评分标准

| 评分 | 说明 |
|------|------|
| ⭐⭐⭐⭐⭐ | 必备，日常高频使用 |
| ⭐⭐⭐⭐ | 强烈推荐，实用性高 |
| ⭐⭐⭐ | 推荐，特定场景好用 |
| ⭐⭐ | 可选，偶尔用到 |
| ⭐ | 实验性，待观察 |

## 代码规范

- 所有文件使用 UTF-8 编码
- Markdown 文件使用 LF 换行
- JSON 文件格式化（2 空格缩进）

## 提交规范

```
feat: 添加新 skill 推荐
fix: 修复信息错误
docs: 更新文档
chore: 更新元数据
```

## 联系方式

如有问题，欢迎在 GitHub Issue 中讨论。
