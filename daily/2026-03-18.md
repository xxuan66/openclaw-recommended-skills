## ⭐ 今日理由

为什么今天推荐 tavily-search？

1. **AI 优化** - 专为 LLM 设计的搜索 API，返回结构化结果
2. **结果精准** - 自动过滤噪音，直接给出相关内容
3. **速度快** - 比传统搜索快 2-3 倍
4. **配合 Agent** - 与 OpenClaw Agent 完美配合，研究场景首选

---

## 🚀 快速安装

```bash
clawhub install tavily-search
```

**配置时间：** 3 分钟（需要 API Key）  
**难度：** ⭐⭐ 简单

---

## 💡 使用示例

```bash
# 深度研究搜索
openclaw agent -m "搜索最新的 RAG 技术论文"

# 快速问答
openclaw agent -m "2026 年 AI Agent 发展趋势是什么？"
```

**💡 技巧：** `tavily-search` 适合需要深度研究的场景，日常简单搜索用 `searxng` 即可。
