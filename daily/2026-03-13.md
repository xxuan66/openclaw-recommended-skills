## ⭐ 今日理由

为什么今天推荐 searxng？

1. **完全免费** - 无需 API Key，自建实例无配额限制
2. **隐私保护** - 不记录搜索历史，不追踪用户
3. **多源聚合** - 同时搜索 Google、Bing、DuckDuckGo 等
4. **日常刚需** - 每天都要用，使用频率最高
