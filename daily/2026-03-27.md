## 🎯 一句话推荐

**给 Agent 装一个结构化大脑，记忆不再是碎片而是知识库**

---

## ⭐ 今日理由

为什么今天推荐 starmemo？

1. **结构化记忆** — 不只是存文字，而是按主题、时间、关联度组织记忆，检索效率高
2. **知识库模式** — 支持笔记 + 标签 + 链接，像个人 Wiki 一样管理 Agent 的知识
3. **互补 self-improving-agent** — self-improving-agent 侧重"反思教训"，starmemo 侧重"知识沉淀"，两者搭配效果更佳

---

## 🚀 快速安装

```bash
clawhub install starmemo
```

**配置时间：** 2 分钟  
**难度：** ⭐⭐ 较简单

---

## 💡 使用示例

```bash
# 让 Agent 记录一条知识
openclaw agent -m "用 starmemo 记录：Python 的 GIL 是全局解释器锁，限制了多线程并行"

# 查询记忆
openclaw agent -m "用 starmemo 查找关于 Python 的记忆"

# 建立关联笔记
openclaw agent -m "用 starmemo 将 GIL 和多线程笔记关联起来"
```

**典型输出：**
- ✅ 已记录到「编程 > Python」分类
- 🔗 发现 3 条相关记忆
- 📝 建议补充：GIL 在 Python 3.13+ 的变化

---

## 🆚 对比：starmemo vs self-improving-agent

| 维度 | starmemo | self-improving-agent |
|------|----------|---------------------|
| **核心功能** | 知识存储与检索 | 反思与自我改进 |
| **记忆方式** | 结构化笔记 | 经验教训 |
| **适合场景** | 积累专业知识 | 避免重复犯错 |
| **搭配使用** | ✅ 强烈推荐 | ✅ 强烈推荐 |

**💡 最佳实践：** 两个都装，starmemo 负责"记住知识"，self-improving-agent 负责"吸取教训"。
