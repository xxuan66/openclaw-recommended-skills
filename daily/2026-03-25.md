## ⭐ 今日理由

为什么今天推荐 skill-vetter？

1. **安全刚需** — 新手最容易忽略的一步，安装前审查能避免泄露 API Key 等风险
2. **一键审查** — 一条命令扫描权限、可疑代码和敏感数据模式
3. **社区信任** — ClawHub 上的 Skill 质量参差不齐，vetter 是你的第一道防线

---

## 🚀 快速安装

```bash
clawhub install skill-vetter
```

**配置时间：** 1 分钟  
**难度：** ⭐ 简单

---

## 💡 使用示例

```bash
# 安装前审查一个 Skill
clawhub vet <skill-name>

# 审查本地 Skill 目录
openclaw agent -m "审查 skills/my-custom-skill 目录的安全性"
```

**典型输出：**
- ✅ 权限范围合理
- ⚠️ 发现硬编码的 API Key 建议移除
- ❌ 检测到可疑的外连请求
