## 🔧 管理命令

### 查看任务状态

```bash
openclaw cron list | grep -E "(模型|mimo|qwen)"
```

### 手动切换模型

```bash
# 切换到 mimo
openclaw models set mimo/mimo-claw-0301

# 切换到 qwen
openclaw models set bailian/qwen3.5-plus

# 查看当前模型
openclaw config get agents.defaults.model.primary
```

### 暂停/恢复任务

```bash
# 暂停切换任务
openclaw cron disable --id <task-id>

# 恢复切换任务
openclaw cron enable --id <task-id>
```

### 删除任务

```bash
openclaw cron remove --id <task-id>
```
