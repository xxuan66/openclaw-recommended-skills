# 💚 微信公众号技能

## 描述
公众号全流程 — 选题、写作、排版、发布，一条龙搞定。支持 Markdown 一键发公众号草稿箱，排版、配图、发布全自动。

## 功能
- **选题策划**: 根据热点、关键词生成公众号选题建议
- **文章写作**: 撰写符合公众号风格的文章内容
- **自动排版**: 将 Markdown 转换为公众号精美排版
- **配图生成**: 自动为文章生成或匹配合适的配图
- **草稿发布**: 一键发布到微信公众号草稿箱

## 使用方法
```
/wechat write <主题>        # 撰写公众号文章
/wechat draft <文章路径>     # 排版并发布到草稿箱
/wechat topics <关键词>      # 获取选题建议
```

## 配置项
- `WECHAT_APP_ID`: 微信公众号 AppID
- `WECHAT_APP_SECRET`: 微信公众号 AppSecret
- `WECHAT_ACCESS_TOKEN`: 微信公众号 access token（可自动获取）

## 输出格式
- 排版后的 HTML 内容
- 草稿箱文章 ID
- 发布状态确认

## 注意事项
- 需要微信公众号官方账号权限
- 图片需符合微信公众号尺寸要求（建议宽度 1080px）
- 发布前请在草稿箱预览确认
