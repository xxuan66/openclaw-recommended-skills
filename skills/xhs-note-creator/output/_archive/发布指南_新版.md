## 📤 发布步骤

### 方式一：手动发布（推荐）

1. 打开小红书网页版：https://www.xiaohongshu.com
2. 点击"发布笔记"
3. 上传所有图片（封面 +12 张卡片）
4. 标题：`OpenClaw 技术解析`
5. 正文：复制 `发布文案.txt` 内容
6. 点击发布 ✅

### 方式二：使用发布脚本

需要先启动 xhs-api 服务：

```bash
# 1. 启动 API 服务
cd xhs-api
python app_full.py

# 2. 发布笔记
python scripts/publish_xhs.py \
  --title "OpenClaw 技术解析" \
  --md content_openclaw_tech.md \
  --images output/cover.png output/card_*.png \
  --api-mode
```
