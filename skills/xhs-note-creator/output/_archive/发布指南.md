## 📱 手动发布步骤

### 方式一：网页版发布

1. 打开小红书创作平台：https://creator.xiaohongshu.com/
2. 点击「发布笔记」
3. 上传所有图片（先选 cover.png 作为封面）
4. 填写标题和正文
5. 添加话题标签
6. 点击发布

### 方式二：手机 App 发布

1. 将图片传到手机
2. 打开小红书 App
3. 点击底部「+」发布
4. 选择所有图片
5. 填写标题和正文
6. 添加标签后发布

## ⚠️ 自动发布说明

当前系统 Python 版本 (3.6) 不支持 `xhs` 库，自动发布功能暂不可用。

如需启用自动发布，需要：
1. 升级 Python 到 3.8+
2. 运行：`pip install xhs`
3. 确保 Cookie 有效
