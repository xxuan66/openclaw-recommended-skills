## 常用命令速查

### 基础渲染
```bash
# 默认设置
python scripts/render_xhs.py content.md

# 自动切分分页（推荐）
python scripts/render_xhs.py content.md -m auto-split
```

### 切换主题
```bash
# 专业商务风（科技、职场内容）
python scripts/render_xhs.py content.md -t professional

# 活泼几何风（生活、美食）
python scripts/render_xhs.py content.md -t playful-geometric

# 植物园风（美妆、护肤）
python scripts/render_xhs.py content.md -t botanical

# 终端风（技术、编程）
python scripts/render_xhs.py content.md -t terminal
```

### 自定义输出
```bash
# 指定输出目录
python scripts/render_xhs.py content.md -o ~/workspace/output/

# 提高清晰度
python scripts/render_xhs.py content.md --dpr 3

# 自定义宽度
python scripts/render_xhs.py content.md -w 1200
```

## 内容创作模板

### 模板 1：产品推荐类

```
标题：[数字] 个让 [结果] 的 [产品类别]！

正文结构：
【引入】痛点/场景描述
【产品 1】名称 + 核心功能 + 使用感受
【产品 2】名称 + 核心功能 + 使用感受
【产品 3】...
【总结】行动号召
【标签】3-5 个关键词
```

### 模板 2：教程分享类

```
标题：[时间] 学会 [技能]！[人群] 必看

正文结构：
【引入】学习这个技能的好处
【步骤 1】详细说明 + 注意事项
【步骤 2】详细说明 + 注意事项
【步骤 3】...
【总结】鼓励行动
【标签】技能相关关键词
```

### 模板 3：经验总结类

```
标题：从 [起点] 到 [终点]，我做了 [数字] 件事

正文结构：
【引入】前后对比，制造反差
【经验 1】具体做法 + 效果
【经验 2】具体做法 + 效果
【经验 3】...
【总结】金句收尾
【标签】成长相关关键词
```

## 常见问题

**Q：图片生成失败？**
A：检查 Python 环境，确保安装了依赖：
```bash
cd /home/admin/.openclaw/workspace/skills/xhs-note-creator
pip install -r requirements.txt
```

**Q：文字显示不全？**
A：使用 `-m auto-fit` 自动缩放或 `-m auto-split` 自动分页

**Q：如何更换字体/样式？**
A：修改 `assets/styles.css` 文件

**Q：可以批量处理吗？**
A：可以，写脚本循环调用渲染命令
