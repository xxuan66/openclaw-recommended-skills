## 📝 内容文件管理

### content/ 目录

存放**已完成**、**可发布**的内容草稿。

**命名规范：**
```
content_<主题>.md
```

**示例：**
- `content_5ai_tools.md` - 5 个 AI 工具推荐
- `content_openclaw.md` - OpenClaw 介绍
- `content_product_launch.md` - 产品发布

### drafts/ 目录

存放**正在编辑**、**未完成**的草稿。

**命名规范：**
```
draft_<主题>_v1.md
draft_<主题>_v2.md
```

**示例：**
- `draft_new_product_v1.md` - 新产品草稿 v1
- `draft_tutorial_v2.md` - 教程草稿 v2

## 📤 输出文件管理

### output/ 目录

**自动按时间戳组织**，每次渲染创建新子目录。

**目录格式：**
```
output/YYYY-MM-DD_HH-mm-ss/
```

**示例：**
```
output/
├── 2026-03-05_12-38-45/    # 2026 年 3 月 5 日 12:38:45 渲染
│   ├── cover.png
│   ├── card_1.png
│   ├── card_2.png
│   ├── card_3.png
│   └── caption.txt
├── 2026-03-05_14-22-10/    # 2026 年 3 月 5 日 14:22:10 渲染
└── 2026-03-06_09-15-30/    # 2026 年 3 月 6 日 09:15:30 渲染
```

### 优点

1. **不会覆盖** - 每次渲染都有独立目录
2. **易于追溯** - 时间戳清晰
3. **版本管理** - 可以保留多次渲染结果
4. **方便对比** - 不同版本的输出可以对比

## 🚀 使用流程

### 1. 创建内容草稿

```bash
# 在 drafts/ 目录创建新草稿
vim drafts/draft_new_topic_v1.md
```

### 2. 编辑完善

```bash
# 反复修改完善
vim drafts/draft_new_topic_v1.md
vim drafts/draft_new_topic_v2.md
```

### 3. 移动到 content/

```bash
# 完成后移动到 content/
mv drafts/draft_new_topic_v2.md content/content_new_topic.md
```

### 4. 渲染输出

```bash
cd /home/admin/.openclaw/workspace/skills/xhs-note-creator

# 方式 1：自动创建时间戳目录（推荐）
node scripts/render_xhs_v2.js content/content_new_topic.md

# 方式 2：指定输出目录
node scripts/render_xhs_v2.js content/content_new_topic.md -o ./output/2026-03-05-new-topic

# 方式 3：从当前目录直接渲染（向后兼容）
node scripts/render_xhs_v2.js ./my-note.md -o ./output/test
```

### 5. 查看输出

```bash
# 查看最新输出
ls -lt output/ | head

# 查看特定输出
ls -la output/2026-03-05_12-38-45/
```

### 6. 发布笔记

```bash
# 进入输出目录
cd output/2026-03-05_12-38-45/

# 发布（自动读取 caption.txt）
source ../../.venv/bin/activate
python ../../scripts/publish_xhs.py -t "笔记标题" -i cover.png card_*.png
```

## 🗂️ 清理旧输出

### 查看输出目录大小

```bash
du -sh output/*/
```

### 删除旧输出（保留最近 10 个）

```bash
cd output/
ls -dt */ | tail -n +11 | xargs rm -rf
```

### 删除特定日期输出

```bash
rm -rf output/2026-03-01_*/
```

## 🔧 自动化脚本

### 清理脚本（可选）

创建 `scripts/cleanup_output.sh`：

```bash
#!/bin/bash
# 保留最近 10 个输出目录
cd "$(dirname "$0")/../output"
ls -dt */ | tail -n +11 | xargs rm -rf
echo "✅ 已清理旧输出，保留最近 10 个"
```

### 发布脚本（可选）

创建 `scripts/publish_latest.sh`：

```bash
#!/bin/bash
# 发布最新渲染的输出
cd "$(dirname "$0")/../output"
LATEST=$(ls -t | head -1)
cd "$LATEST"
source ../../.venv/bin/activate
python ../../scripts/publish_xhs.py -t "$1" -i cover.png card_*.png
```
