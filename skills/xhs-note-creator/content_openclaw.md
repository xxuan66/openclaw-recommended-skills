# 🚀 什么是 OpenClaw？

开源 AI 助理框架，让你的 AI 会干活！

> 不只是聊天，还能帮你发邮件、做图、发小红书！

**核心能力：**
- 💬 多平台消息（微信/Telegram/钉钉）
- 🛠️ 技能系统（自定义扩展）
- 📅 定时任务（自动提醒）
- 🌐 网页搜索（隐私优先）

---

# ⚡ 快速开始

3 步拥有你的 AI 助理：

```bash
# 1. 安装
npm install -g openclaw

# 2. 配置
openclaw configure

# 3. 启动
openclaw start
```

**支持平台：**
- 📱 移动端：iOS/Android
- 💻 桌面：Windows/Mac/Linux
- 🌐 网页版：随时随地访问

# 🎯 核心功能

## 1️⃣ 技能系统

像装 App 一样给 AI 加能力：

```bash
# 从 ClawHub 安装技能
clawhub install xhs-note-creator
clawhub install weather
clawhub install feishu-doc
```

## 2️⃣ 记忆系统

AI 记得你说过的话：
- 📝 每日笔记自动保存
- 🧠 长期记忆持久化
- 🔍 快速检索历史

## 3️⃣ 自动化

定时任务 + 触发器：
- ⏰ 每天早上 9 点天气提醒
- 📧 新邮件自动通知
- 📊 定时数据报告
