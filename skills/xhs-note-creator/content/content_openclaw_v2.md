# 🚀 什么是 OpenClaw？

开源 AI 助理框架，让你的 AI 真正会干活！

不只是聊天，还能：
✅ 自动发邮件
✅ 生成图片卡片
✅ 发布小红书
✅ 定时提醒

---

# ⚡ 快速开始

```bash
# 1. 安装
npm install -g openclaw

# 2. 配置
openclaw configure

# 3. 启动
openclaw start
```

支持：Windows / Mac / Linux

**技能系统** - 像装 App 一样扩展能力
