# 🧑 AI 文本人性化技能

## 描述
去除 AI 痕迹，让文字更自然。6 种风格变换，16 种检测模式。

## 功能
- **AI 痕迹检测**: 识别文本中的 AI 生成特征
- **人性化改写**: 将 AI 文本改写为自然人类风格
- **风格变换**: 支持 6 种不同写作风格
- **多轮优化**: 迭代优化直到达到目标效果
- **检测模式**: 16 种 AI 检测器验证
- **批量处理**: 支持多文档批量人性化

## 使用方法
```
/humanize <文本>            # 人性化改写文本
/humanize check <文本>      # 检测 AI 痕迹
/humanize style <文本> <风格> # 指定风格改写
/humanize batch <文件路径>   # 批量处理
```

## 支持风格
1. **自然 casual**: 日常对话风格
2. **专业 professional**: 商务专业风格
3. **创意 creative**: 富有创意的表达
4. **学术 academic**: 学术论文风格
5. **幽默 humorous**: 轻松幽默风格
6. **简洁 concise**: 简洁明了风格

## 配置项
- `HUMANIZER_MODEL`: 人性化模型选择
- `HUMANIZER_STYLE`: 默认风格
- `HUMANIZER_STRENGTH`: 改写强度 (1-10)
- `DETECTOR_API_KEY`: AI 检测器 API 密钥

## 输出格式
- 人性化后的文本
- AI 检测分数对比（改写前后）
- 风格分析报告

## 注意事项
- 改写强度越高，原文保留越少
- 建议人工审核最终输出
- 某些专业内容可能需要保留术语
