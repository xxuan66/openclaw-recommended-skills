## 📊 xiaohongshu-mcp 详细功能

### 1️⃣ 搜索热门笔记

```bash
# 搜索关键词
python scripts/xhs_client.py search "关键词"

# 搜索并保存结果
python scripts/xhs_client.py search "AI 工具" > search_results.txt
```

**输出内容：**
- 笔记 ID
- 标题
- 描述
- 点赞数
- 收藏数
- 评论数
- xsec_token（用于获取详情）

### 2️⃣ 获取笔记详情

```bash
python scripts/xhs_client.py detail "note_id" "xsec_token"
```

**输出内容：**
- 完整正文
- 所有图片 URL
- 全部评论
- 用户信息
- 发布时间
- 互动数据

### 3️⃣ 查看推荐 Feed

```bash
python scripts/xhs_client.py feeds
```

**用途：**
- 了解当前热门话题
- 发现 trending 内容
- 获取创作灵感

### 4️⃣ 发布笔记

```bash
python scripts/xhs_client.py publish "标题" "正文内容" "图片 URL1,图片 URL2"
```

**注意：** 图片需要是先上传到图床的 URL

## 🔐 登录配置（Linux 服务器）

### 问题
`xiaohongshu-mcp` 需要扫码登录，但 Linux 服务器无图形界面。

### 解决方案

#### 方案 A：本地登录，复制 Cookie（推荐）

**1. 在本地电脑（有浏览器）登录：**

```bash
# macOS
brew install xiaohongshu-mcp
xiaohongshu-mcp login

# Windows
# 下载 https://github.com/xpzouying/xiaohongshu-mcp/releases
xiaohongshu-mcp-login.exe
```

**2. 获取 Cookie：**

登录后会生成配置文件：
```bash
# macOS/Linux
cat ~/.xiaohongshu-mcp/config.json

# Windows
cat %USERPROFILE%\.xiaohongshu-mcp\config.json
```

**3. 复制到服务器：**

```bash
# 方法 1：直接复制配置
scp ~/.xiaohongshu-mcp/config.json user@server:~/.xiaohongshu-mcp/

# 方法 2：提取 Cookie 手动配置
# 从 config.json 中提取 cookie 字段
# 添加到服务器的 .env 文件
```

#### 方案 B：使用浏览器插件获取 Cookie

**1. 在服务器浏览器登录小红书：**

```bash
# 如果有浏览器
firefox https://www.xiaohongshu.com
```

**2. 获取 Cookie：**

- F12 打开开发者工具
- Network 标签
- 刷新页面
- 查看请求的 Cookie
- 复制完整 Cookie 字符串

**3. 配置到 .env：**

```bash
# 编辑 xiaohongshu-mcp 的配置
vim ~/.xiaohongshu-mcp/config.json

# 添加 Cookie
{
  "cookie": "acw_tc=...; a1=...; web_session=..."
}
```

#### 方案 C：使用 xhs 库的 Cookie（最简单）

**直接用你现在 `xhs-note-creator` 的 Cookie：**

```bash
# 你的 Cookie 已经在这里
cat /home/admin/.openclaw/workspace/skills/xhs-note-creator/.env

# 复制到 xiaohongshu-mcp
cp /home/admin/.openclaw/workspace/skills/xhs-note-creator/.env \
   /home/admin/.openclaw/workspace/skills/xiaohongshu-mcp/.env
```

## ❓ 常见问题

### Q: xiaohongshu-mcp 登录失败？
**A:** 检查 Cookie 是否过期，从浏览器重新复制。

### Q: 搜索结果为空？
**A:** 可能是关键词太冷门，换个热门词试试。

### Q: 发布失败？
**A:** 检查图片 URL 是否可访问，或用 xhs-note-creator 发布。

### Q: 两个技能怎么选？
**A:** 
- 创作发布 → xhs-note-creator
- 调研分析 → xiaohongshu-mcp
