# 🤖 基础智能包技能

## 描述
基础 AI 能力集合，包含对话理解、文本处理、简单任务自动化等核心功能。

## 功能
- **智能对话**: 自然语言理解和回复
- **文本处理**: 翻译、总结、改写、润色
- **任务自动化**: 简单重复任务自动化
- **知识问答**: 基于已有知识回答问题
- **文件处理**: 读取、分析、转换文件格式
- **提醒设置**: 创建一次性或周期性提醒

## 使用方法
```
/ask <问题>               # 提问获取答案
/translate <文本> <目标语言> # 翻译文本
/summarize <文本/文件>     # 生成摘要
/rewrite <文本> <风格>     # 改写文本
/remind <时间> <内容>      # 设置提醒
/process <文件>           # 处理文件
```

## 配置项
- `DEFAULT_MODEL`: 默认 AI 模型
- `DEFAULT_LANGUAGE`: 默认语言
- `MEMORY_ENABLED`: 是否启用记忆功能
- `REMINDER_CHANNEL`: 提醒通知渠道

## 输出格式
- 文本回答
- 处理结果
- 任务执行状态

## 注意事项
- 复杂任务建议拆分为多步
- 重要操作前建议确认
- 定期清理记忆避免冗余
