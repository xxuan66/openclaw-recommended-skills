# 🖼️ AI 图片生成技能

## 描述
一句话出图 — 10+ 模型随心选，从产品图到艺术画，AI 秒级交付。支持文生图、图生图、风格迁移、局部重绘。

## 功能
- **文生图**: 根据文字描述生成图片
- **图生图**: 基于参考图生成新图片
- **风格迁移**: 将图片转换为特定艺术风格
- **局部重绘**: 修改图片的特定区域
- **高清放大**: 提升图片分辨率和质量
- **批量生成**: 一次生成多张变体供选择

## 使用方法
```
/img gen <描述>             # 文生图
/img edit <图片> <描述>      # 图生图/编辑
/img style <图片> <风格>     # 风格迁移
/img inpaint <图片> <区域>   # 局部重绘
/img upscale <图片>         # 高清放大
```

## 支持模型
- Stable Diffusion XL
- DALL-E 3
- Midjourney (通过 API)
- Flux
- Playground AI
- 通义万相
- 其他 10+ 模型

## 配置项
- `IMAGE_GEN_PROVIDER`: 默认图片生成服务商
- `IMAGE_GEN_API_KEY`: API 密钥
- `IMAGE_GEN_MODEL`: 默认模型
- `IMAGE_GEN_SIZE`: 默认尺寸 (1024x1024, 1920x1080 等)
- `IMAGE_GEN_STYLE`: 默认风格

## 输出格式
- 生成的图片文件路径
- 图片预览链接
- 生成参数记录

## 注意事项
- 不同模型有不同的最佳实践
- 建议生成多张变体后选择
- 注意版权和使用限制
