# 📄 文档/内容类 Skill 推荐

## summarize

**版本:** 1.0.0  
**Slug:** `summarize`  
**类别:** 文档/内容  

### 简介
使用 summarize CLI 摘要 URL 或文件（支持 web、PDF、图片、音频、YouTube）。

### 推荐理由
- ✅ 多格式支持
- ✅ 一键摘要
- ✅ 节省时间
- ✅ 支持 YouTube

### 安装命令
```bash
clawhub install summarize
```

### 配置要求
- 无需额外配置

### 支持格式
| 格式 | 支持 |
|------|------|
| Web URL | ✅ |
| PDF | ✅ |
| 图片 | ✅ |
| 音频 | ✅ |
| YouTube | ✅ |

### 使用示例
```bash
# 摘要网页
summarize https://example.com

# 摘要 PDF
summarize document.pdf

# 摘要 YouTube 视频
summarize https://youtube.com/watch?v=xxx
```
